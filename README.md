# American Energy Money

> The Intersection of Value and Science

Static site rebuilt from [americanenergymoney.com](https://www.americanenergymoney.com).

## Structure

```
/
├── index.html        ← main page
├── images/           ← all site images (extracted from original)
└── README.md
```

## Local preview

```bash
npx serve .
# or
python3 -m http.server 8080
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```
